Basis HTML

Wat is HTML?

HTML is een programmeertaal waarmee je websites kunt bouwen. De afkorting van HTML is Hypertext Markup Language.

1. Basis HTML

Je opent een leegbestand HTML editor (sublime of Visual Studio Code etc.) En je begint eerst met de opbouw, dat gaat als volgt:

  1. html
  2. head
  3. title
  4. /head (sluiten)
  5. body
  6. /body (sluiten)
  7. /html (sluiten)

Je slaat het bestand op als index.html, dit wilt zeggen dat de HTML-pagina als eerst zal worden gezocht door een browser. Nadat je het bestand met de opbouw opgeslagen hebt kun je het openen in de browser. Wat je dan te zien krijgt is een lege pagina.

2. HTML tags

De pagina begint met < html > en eindigt met < / html >, alles tussen deze tags is HTML-code. De HTML-elementen zijn omsloten door tags en te herkennen aan punthaken. Binnen de html tags komen de header en de body van de pagina. Alles tussen deze tags < body > < / body > wordt getoond in het browservenster.

3. Header van de pagina

De header is het hoofd van de pagina, hierin verwerk je de titel van de pagina. De header begint met < head > en eindigt met < / head >. Alles tussen deze tags bestaat uit zogenaame metagegevens, zoals een paginatitel.

4. Titel van de pagina

In de header staat de titel van de pagina, deze zet je tussen < title >Page Title< / title >. Tussen deze tags wordt de titel van de webpagina bepaald.

5. Starten met HTML

Nu de opbouw van de html uitgetypt is hoef je het alleen nog aan te vullen met een hoofdstukaanduiding daarvoor gebruik je de tag < h1 > < / h1> . Vervolgens wil je ook tekst onder de hoofdstuk (paragraaf) hebben, hiervoor gebruik je < p >Test< / p > alles tussen deze tags is tekst binnen een paragraaf.
